<h2>🌌 About The Project</h2>

<p>
SolveIt is a next-generation problem-solving marketplace where users can post real-world challenges, 
and verified experts compete to solve them.
</p>

<p>
Designed with an <b>Anti-Gravity AI interface</b>, the platform creates a floating, immersive experience 
that feels modern, fluid, and intelligent.
</p>

---

<h2>⚡ Core Features</h2>

<ul>
  <li>📝 Post Problems with budget & deadline</li>
  <li>🤝 Experts bid to solve problems</li>
  <li>💬 Real-time communication</li>
  <li>💳 Secure payment system</li>
  <li>📁 File attachment support</li>
  <li>⭐ Rating & review system</li>
</ul>
